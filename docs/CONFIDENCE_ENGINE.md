# Confidence Engines

Confidence analysis exists in both the reasoning and reflection layers of JARVIS OS.

`ConfidenceEngine` in the Reasoning layer estimates how confident the executive system should be before execution.

`ReflectionConfidence` in the Reflection layer measures how well that confidence matched the final outcome after execution.

## Measures

- Expected confidence
- Actual confidence
- Confidence error
- Confidence drift
- Decision reliability
- Planning reliability
- Reasoning reliability
- Retrieval reliability
- Research reliability
- Workflow reliability
- Learning reliability

## Output

- Confidence report
- Reliability report
- Improvement recommendations

This is advisory analysis only. It does not change Executive JARVIS decisions automatically.
