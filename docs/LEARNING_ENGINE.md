# Reflection Learning

`ReflectionLearning` captures learning metadata after execution so future executive reasoning can improve.

## Captured Pattern Types

- Successful and failed decisions
- Successful and failed plans
- Successful and failed workflows
- Retrieval, research, task, and conversation patterns
- User preferences
- Executive decisions

## Boundary

This component stores metadata only. Persistent storage remains the responsibility of the existing Memory and Knowledge engines.
